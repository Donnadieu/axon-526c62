# axon

Axon is a fully client-side SvelteKit application using adapter-static that renders tasks in a 3D spatial grid via Threlte (Three.js for Svelte). All data persists in a single human-readable markdown file accessed through the File System Access API, with an import/export fallback for unsupported browsers. No backend, no database, no auth — pure local-first static SPA.
